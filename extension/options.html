Website Access to Bluetooth Devices

You can view and manage website access to devices here.

    Install WebBT Server

    WebBT requires the the WebBT server application to be installed on your system. Please ensure you have installed the current version of WebBT server.

    Compatibility

    WebBT Server is compatible with all modern versions of Windows 10 (version 1703+) and Windows 11.
