Install WebBT Server

This page is being shown because a page tried to use Web Bluetooth functionality but the WebBT server application isn't installed or an incompatible version is installed. Please install the current version of WebBT server, ensure the WebBT extension is up-to-date, and try again.

Compatibility

WebBT Server is compatible with all modern versions of Windows 10 (version 1703+) and Windows 11.
